# 移植文档.md

## 1.简介

CppQtFluentUi:采用C++对PyQt-Fluent-Widgets进行重新编写，用以支持C++版的Qt的FluentUi，由于项目较为复杂，仅以该文档描述已移植的相关内容。方便检索及后续代码编写。
